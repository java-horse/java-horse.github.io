MybatisPlus动态添加切换数据源

MyabtisPlus集成Druid连接池，实现数据源的动态添加、删除、切换等操作。主要利用@DS注解实现动态切换。

原理步骤：

  1. 在master主数据源库创建一个管理其他数据源的信息表，用于存储datasource的必要信息
  2. 当接收请求时，根据请求的自定义唯一标识，到MySQL读取数据源信息并动态的添加数据源，一定程度上利用Redis进行缓存。
  3. 利用MybatisPlus的@DS注解进行动态切换数据源
  4. 根据一定的算法，例如lru算法（最少用的数据源），进行删除数据源。
